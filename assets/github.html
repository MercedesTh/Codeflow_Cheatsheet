  • pwd

    Where am I? Whose computer is this?

  • ls

    Lists all existing data

  • ls -1

    Lists all existing data in a list = Orientation

  • clear

    Creates a blank page by pushing all terminal content up

  • ls -l

    Lists all data with specific information

  • ls -a

    Lists hidden data (which we couldn't see otherwise on a Mac)

  • cd example

    Changes directory to "example"

  • cd ..

    Changes directory to the previous one

  • cd ../..

    Changes two directories backwards

  • mkdir example

    Creates new directory "example"

  • rmdir example

    Removes directory "example" irrevocably

  • touch example

    Creates new file "example"

  • rm example

    Removes file "example" irrevocably

  • mv example example/example

    Moves directory or file to "example/example"

  • cp example

    Copies directory or file "example"

  • cp example new name

    Copies directory or file giving it a "new name"

  • dig

    Asks for domain name system

  • ~

    Redirects to the user's main directory

  • exit

    Closes terminal
