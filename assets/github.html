Cheat
  • git init

    Initiates the linking between your draft and a new repository. A .git file is hidden in your directory.

  • git add .

    Adds draft / new data for an existing repository to the staging area

  • git commit -m "example"

    Commits a message / title for the draft / new data to the staging area. Don't forget to copy & paste the three last lines from your repository just created on GitHub. Enter!

  • git push

    Pushes draft / new data to an existing repository from the staging area to the platform GitHub

  • clear

    Creates a blank page by pushing all terminal content up

  • git checkout -b example

    Creates a new branch named example to the repository. Procede with git add., git commit -m "", git push. Don't forget to copy & paste the penultimate line from the error message. Enter!

  • git checkout example

    Goes to the branch named example to add new data. Procede with git add., git commit -m "", git push.

  • git pull

    Pulls the newest version of the repository from GitHub, including all accepted & merged changes that have been made to main and branches

  • git pull origin main

    Pulls the newest version of the main branch in the repository while not all changes / branches have been accepted & merged

  • git status

    Asks for the actual status of the rpository
